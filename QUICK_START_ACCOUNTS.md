# 🚀 Quick Start: Multi-Account Management

## เริ่มใช้งานใน 3 ขั้นตอน

### 1️⃣ เปิดหน้า Accounts
```
เปิดเว็บ → กด 👥 Accounts tab
```

### 2️⃣ เพิ่ม Account
```
กด ➕ เพิ่ม Account
→ กรอก: ชื่อ, Client ID, Client Secret
→ กด บันทึก
```

**ดึง Client ID & Secret ได้จาก:**
https://console.cloud.google.com/apis/credentials

### 3️⃣ Login YouTube
```
กด 🔐 Login YouTube ที่ account ที่ต้องการ
→ เลือก Google Account
→ อนุญาต YouTube access
→ กลับมาหน้า Accounts
```

---

## ✅ พร้อมใช้งานแล้ว!

- **สลับ account**: กด "✅ ใช้ Account นี้"
- **อัปโหลด**: จะใช้ account ที่เป็นสีเขียว (active)
- **Quota**: แต่ละ account มี 10,000 units/day (6 videos/day)

---

## 💡 ประโยชน์

1. **หลาย YouTube channels** - จัดการหลายช่องในที่เดียว
2. **ไม่โดน quota limit** - แต่ละ account มี quota แยกกัน
3. **ใช้งานง่าย** - สลับ account ด้วยคลิกเดียว
4. **Multi-user** - หลายคนใช้ระบบร่วมกัน

---

## 🎯 Example

```
Account A: Main Channel (6 videos/day)
Account B: Backup Channel (6 videos/day)
Account C: Client Channel (6 videos/day)

Total: 18 videos/day! 🚀
```
