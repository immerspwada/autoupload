# 🎬 YouTube Auto Uploader - ฟีเจอร์ที่ใช้งานได้

## ✅ ฟีเจอร์หลัก (11 ฟีเจอร์)

### 1. 📊 **Dashboard**
**หน้าภาพรวมระบบ**
- แสดงสถิติการอัปโหลด (วันนี้/7 วันล่าสุด)
- Quota status (เหลือกี่วิดีโอ)
- Queue status (กำลังอัป/รอ)
- Upload chart
- Recent uploads

**ใช้งาน:** เปิดแล้วเห็นสถานะทั้งหมดในที่เดียว

---

### 2. 📤 **Upload**
**อัปโหลดวิดีโอไป YouTube**
- อัปเดี่ยว (single upload)
- อัปทั้งหมด (batch upload)
- ตั้งค่า: title, description, tags, privacy
- แสดง progress bar

**ใช้งาน:** 
1. เลือกไฟล์จากโฟลเดอร์
2. กรอก title/description
3. กดอัปโหลด

---

### 3. 🎵 **TikTok**
**ดาวน์โหลด + อัปโหลด TikTok → YouTube**
- ค้นหา TikTok ด้วย keyword
- แสดง virality score (🔥/📈/👍/📉)
- Monetization check (✓/⚠️/🚫)
- SEO preview
- Batch download + upload
- Smart quota filtering

**ใช้งาน:**
1. ใส่ keyword (เช่น "แมว")
2. เลือกคลิปที่ต้องการ
3. กด "Download & Upload"

**ฟีเจอร์พิเศษ:**
- ✅ No watermark (ไม่มีโลโก้ TikTok)
- ✅ Auto SEO (สร้าง title/tags/category)
- ✅ Virality scoring (เลือกคลิปดีก่อน)
- ✅ Monetization safety (บล็อกคลิปเสี่ยง)

---

### 4. 👥 **Accounts**
**จัดการหลาย YouTube accounts**
- เพิ่ม/ลบ account
- Login YouTube per account
- สลับ active account
- Quota tracking per account
- แสดง channel info

**ใช้งาน:**
1. กด "➕ เพิ่ม Account"
2. กรอก Client ID & Secret
3. กด "🔐 Login YouTube"
4. กด "✅ ใช้ Account นี้"

**ประโยชน์:**
- หลาย YouTube channels → จัดการในที่เดียว
- แต่ละ account มี quota 10,000 units แยกกัน
- Multi-user friendly

---

### 5. 📁 **Files**
**ดูไฟล์ที่ดาวน์โหลดไว้**
- แสดงไฟล์ใน `downloads/` และ `downloads/tiktok/`
- ข้อมูล: ชื่อไฟล์, ขนาด, วันที่
- อัปโหลดเดี่ยว
- ลบไฟล์

**ใช้งาน:** ดูไฟล์ที่ดาวน์โหลดมา เลือกอัป หรือลบ

---

### 6. 🔄 **Queue**
**ดูคิวการอัปโหลด**
- รายการที่กำลังอัป
- รายการรอคิว
- รายการสำเร็จ/ล้มเหลว
- Progress real-time
- ยกเลิก/ลองใหม่

**ใช้งาน:** ดูสถานะ queue เมื่ออัปแบบ batch

---

### 7. ⏰ **Scheduler**
**ตั้งเวลาอัปโหลดอัตโนมัติ**
- Auto-scan folder ตามเวลาที่กำหนด
- Folder watcher (อัปทันทีเมื่อมีไฟล์ใหม่)
- ตั้งช่วงเวลาสแกน
- เปิด/ปิด scheduler

**ใช้งาน:**
1. เลือกโฟลเดอร์
2. ตั้งช่วงเวลาสแกน (นาที)
3. เปิด scheduler

---

### 8. 💎 **SEO**
**ตั้งค่า SEO auto-optimization**
- โหมด: Auto (สร้าง metadata อัตโนมัติ) / Manual
- Title template
- Channel description (แนบท้ายทุกวิดีโอ)
- Auto-schedule prime-time
- SEO preview

**ใช้งาน:** 
1. เลือกโหมด Auto
2. กรอก channel description
3. เปิด auto-schedule
4. บันทึก

**ผลลัพธ์:**
- Title, tags, category ถูกสร้างอัตโนมัติจาก TikTok caption
- เผยแพร่ช่วง prime-time (19:00-21:00)
- รายได้จากโฆษณาเพิ่มขึ้น

---

### 9. 📋 **Activity**
**ติดตาม real-time event log**
- ทุกเหตุการณ์: upload, download, queue, scheduler, auth
- กรองตาม: ประเภท, ระดับ (success/error/warning)
- สถิติ: ทั้งหมด, วันนี้, สำเร็จ, ผิดพลาด
- ล้างประวัติ

**ใช้งาน:** ดูว่าระบบทำอะไรไปบ้าง debug เมื่อมีปัญหา

---

### 10. 📜 **History**
**ประวัติการอัปโหลด**
- รายการวิดีโอที่อัปโหลดแล้ว
- ข้อมูล: ชื่อไฟล์, วันเวลา, YouTube URL
- ลิงก์ไปดูใน YouTube
- ล้างประวัติ

**ใช้งาน:** ดูว่าอัปอะไรไปบ้าง กดลิงก์ดูใน YouTube

---

### 11. ⚙️ **Settings**
**ตั้งค่าระบบ**
- โฟลเดอร์ดาวน์โหลด
- Privacy default (public/unlisted/private)
- Auto-delete หลังอัป
- Notification settings

**ใช้งาน:** ตั้งค่าพฤติกรรมของระบบ

---

## 🔥 ฟีเจอร์เด่น

### 1. **TikTok Integration** 🎵
- ดาวน์โหลด TikTok (no watermark)
- ค้นหาด้วย keyword
- Virality scoring → เลือกคลิปดีก่อน
- Auto SEO → title/tags/category
- Monetization check → บล็อกคลิปเสี่ยง

### 2. **Multi-Account System** 👥
- จัดการหลาย YouTube accounts
- Quota tracking แยกต่าง account
- สลับง่าย คลิกเดียว

### 3. **Smart Quota Management** 🎯
- Real-time quota tracking
- Auto-reset เที่ยงคืน PST
- Smart filtering (อัปคลิปดีเมื่อ quota น้อย)
- แสดง quota status ชัดเจน

### 4. **SEO Automation** 💎
- Auto-generate title/description/tags
- Prime-time scheduling
- Category detection
- Monetization optimization

## 🗑️ ฟีเจอร์ที่ลบออก (ไม่ใช้งาน)

### ❌ Browser Upload Mode
**เหตุผลที่ลบ:**
- ไม่ได้ทำงานจริง (แค่ placeholder)
- ซับซ้อนเกินไป (ใช้ Puppeteer)
- ช้ากว่า API mode มาก
- API mode + Extended Quota ดีกว่า

**ทางเลือก:**
1. ใช้ API Mode (จำกัด 6 videos/day)
2. ขอ Extended Quota → 1M+ units/day (600+ videos/day)

## 🚀 Quick Start

### 1. เริ่มต้นใช้งาน
```bash
npm start
# เปิด http://localhost:3000
```

### 2. Login YouTube
```
📊 Dashboard → กด "เข้าสู่ระบบ"
```

### 3. อัปโหลดวิดีโอ
```
📤 อัปโหลด → เลือกไฟล์ → กดอัป
```

### 4. ดาวน์โหลด TikTok
```
🎵 TikTok → ใส่ keyword → เลือกคลิป → Download & Upload
```

### 5. เพิ่ม Account (ถ้าต้องการหลาย accounts)
```
👥 Accounts → ➕ เพิ่ม Account → Login
```
